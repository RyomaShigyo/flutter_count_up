# flutter_count_up
デフォルトのカウントアップアプリにカウントダウン、リセット機能、サウンド、アニメーションを追加し、
riverpodを使用してMVVMにしました。データは端末に保存されます。

＜サウンド音源＞
https://otologic.jp/free/se/flash01.html#google_vignette
